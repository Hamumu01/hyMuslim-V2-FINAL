# hyMuslimplus PWA

A Progressive Web App for prayer times and Quran reading, built with React, Vite, and Tailwind CSS.

## Features
- Accurate prayer times by city
- Quran reading with bookmarks
- Installable as a PWA (offline support, home screen)
- Dark mode
- Responsive UI for all devices
- Notification support

## Getting Started

### 1. Install dependencies
```bash
npm install
```

### 2. Run locally
```bash
npm run dev
```

### 3. Build for production
```bash
npm run build
```

### 4. Preview production build
```bash
npm run preview
```

## PWA Features
- Manifest and service worker are pre-configured.
- Add to Home Screen on mobile.
- Offline support for key pages.

## Notes
- For full notification support, allow notifications in your browser.
- Test on multiple devices for best experience.
